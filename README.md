# Binarimage
A tools that convert a text into a binary image, like a QR code btw. 

## Feature
- **Costume color** using rgb (255, 255, 255)

> BTW, this shit has cool hacker animation because, i want it yk. 

## How To Use
### 1. Clone this repo
```bash
git clone https://github.com/leverec/Binarimage.git
cd Binarimage
```
### 2. Run Main Script
```bash
python main.py
```
### 3. Custom color
You can use costum color or default (black & white) If using a **Custom color** you had to enter the **RGB** value of the color you wanna set. 
### 4. Enter Text
Type anything and make sure you don't pass the area limit (128 × 128) but you can change it through changing the code
